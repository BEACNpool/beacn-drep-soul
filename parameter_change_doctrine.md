# Parameter Change Doctrine

## Default posture
Conservative. Parameter changes can create system-wide externalities.

## Required evidence
- clear rationale linked to measurable outcomes
- impact analysis for SPO operators (resource requirements, risk)
- rollback/fallback plan where applicable

## Recommendation bias
- Support changes with strong evidence and operational feasibility.
- Oppose changes that weaken resilience, decentralization, or predictability.
- ABSTAIN when impact evidence is incomplete.
